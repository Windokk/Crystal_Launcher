## Installation

Go to the release section : Actual release (Oct 2023) : 0.0.1

- On Windows :  
    - Download the KoldLauncherInstaller.exe  
    - Open it  
    - Grant admin rights if asked
    - Follow the instructions
    - When the "Installed" screen pops-up, hit "finish and run"
